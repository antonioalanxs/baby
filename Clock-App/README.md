* Basics of [HTML](https://html.spec.whatwg.org/).

  * Dynamic and Weak Typing concepts,
  * Basic syntax,
  * Basics of DOM treatment,
  * Arrow functions,
  * Anonymous functions,
  * Callbacks,
  * OOP,
  * Date Object,
  * Transversal:
    * Ternary operators,
    * Passing parameters by value and reference to functions,
    * Receiving unlimited parameters in a function,
    * Object prototypes.
